# Changelog

All notable changes to Clipboard Snap are documented in this file.

## [0.1.0] - 2026-07-15

### Added

- Signed iOS Shortcut for saving Share Sheet or clipboard text to Turso.
- Parameterized SQL over HTTP request with an insert-only database token.
- First-line success notification preview.
- Deterministic plist generator, structural validator, and signing helper.
- Make targets for public builds, private configured builds, Turso setup, and
  record retrieval.
- Reusable Clipboard Snap skill and open-source project documentation.
